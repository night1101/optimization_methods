# Оптимизация: сравнение градиентных и безградиентных методов

##  Постановка задачи

Найти глобальный минимум функции:
f(x₁, x₂) = (x₁² + 2x₂ - 10)² + (x₁ + x₂² - 5)²

**Особенности функции:**
- Функция имеет **два глобальных минимума** (f ≈ 0)
- Один **локальный минимум** (f ≈ 0)
- Является тестовой функцией для проверки алгоритмов оптимизации

### Известные экстремумы

| Тип | x₁ | x₂ | f(x₁, x₂) |
|-----|----|----|-----------| 
| Глобальный минимум A | 3.525428 | -1.214320 | ≈ 0 |
| Глобальный минимум B | -2.156324 | 2.675131 | ≈ 0 |
| Локальный минимум | 2.630900 | 1.539200 | ≈ 0 |

##  Реализованные методы

### 1. DFP (Davidon-Fletcher-Powell)
**Тип:** Градиентный (квазиньютоновский)

**Особенности:**
- Аппроксимирует обратную матрицу Гессе
- Использует Armijo rule для адаптивного выбора шага
- Требует вычисления градиента

**Формула обновления:**
H_new = H + (s·sᵀ)/(s·y) - (H·y·yᵀ·H)/(yᵀ·H·y)

### 2. Метод Пауэлла (Powell's Method)
**Тип:** Безградиентный (прямой)

**Особенности:**
- Минимизация по сопряжённым направлениям
- Одномерный поиск через золотое сечение
- Не требует вычисления градиента

### 3. CEM (Cross-Entropy Method)
**Тип:** Эвристический (глобальная оптимизация)

**Особенности:**
- Генерация выборок из многомерного нормального распределения
- Элитный отбор лучших решений
- Адаптивное обновление параметров распределения
- Способен находить глобальный минимум

##  Ключевые выводы

1. **DFP** — самый эффективный для локальной оптимизации
   - Требует градиент, но очень быстрый (6-19 итераций)
   - Чувствителен к начальному приближению

2. **Метод Пауэлла** — хорош, когда градиент недоступен
   - Больше вызовов функции, но стабилен
   - Не находит глобальный минимум из произвольной точки

3. **CEM** — единственный метод, способный к глобальному поиску
   - Очень затратный (8000-10000 вызовов)
   - Требует тонкой настройки параметров
